# Sauce Box V1 Scope Lock

## Status

Locked for V1 development.

No additional features are added unless they improve first-use sound quality, stability, or conversion.

## Free (V1) Included

1. Core texture engine.
2. Controls: Drive, Texture, Wow Depth, Wow Rate, Tone, Mix, Output.
3. Output safety chain:
   - Gain compensation behavior tied to drive/mix.
   - Soft output limiting.
4. Starter preset behavior and baseline preset bank (target: 12 presets).
5. Stable standalone and plugin startup on macOS.

## Pro (V1) Target Locks

1. Expanded preset bank and favorites.
2. Advanced shaping modules (transient/stereo/multiband texture).
3. Oversampling (2x/4x).
4. A/B compare.
5. Additional macros and deeper workflow tools.

## Not In Free V1

1. Advanced shaping modules.
2. Oversampling.
3. A/B compare.
4. Extended preset management features.

## Conversion Path (V1)

1. Subtle in-plugin CTA for track review.
2. Link to a short lead form.
3. Follow-up process for service conversion.
